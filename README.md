## Descrição das Variaveis

As variaveis utilizadas no código são:

- `API_URL`: URL da API
- `PORT`: porta do servidor
- `DEBUG`: modo de depuração (true/false)
- `USER`: nome do usuário
- `TOKEN`: token de autenticação
